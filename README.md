Goal is to develop board and esphome config for home assistant voice satelite based for bluetooth speaker from aliexpress.

- Support for 2 microphones (for future)
- Mono aplifier
- Support for 4 hardware buttons present in oryginal product.
- Led ring
- Line out analog output (PCM5102)

home-assistant-voice-usb is a experimental configuratuion for usb conference speaker (emet M1A)

Configuration is based on https://github.com/KristopherMackowiak/ha_voice_assistant
